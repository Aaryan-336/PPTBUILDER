# AI PPT Builder

## Vision

Transform Excel spreadsheets into fully completed PowerPoint presentations using user-uploaded PPT templates.

Users should only upload:

1. PPT Template
2. Excel Spreadsheet

The system should automatically:

- Understand slide intent.
- Analyze spreadsheet data.
- Generate insights.
- Create charts.
- Fill slides.
- Return a downloadable PPTX.

---

## Problem

Business users spend hours manually converting Excel data into PowerPoint presentations.

This process is:

- Repetitive
- Error-prone
- Expensive
- Slow

---

## Users

- Consultants
- Sales teams
- Finance teams
- Analysts
- Agencies
- Management teams

---

## Success Metrics

- PPT generation under 2 minutes.
- 95% template preservation.
- 90% reduction in manual work.
